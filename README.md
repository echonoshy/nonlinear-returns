# nonlinear-returns
关于钱、AI、时间、复利、判断力和经验资产化的思考与复盘。

## Development

This site is a Vite-powered static site. Source pages live in `src/`, static
content lives in `public/`, and production output is written to `dist/`.

```bash
npm install
npm run dev
npm run build
npm run preview
```

GitHub Pages deployment is handled by `.github/workflows/deploy.yml`.
