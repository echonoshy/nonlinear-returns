# 《非线性回报》第一季 · 12 期选题 v3.1

> 五幕弧线:**钱怎么回事(1–3) → 定方向(4–5) → 下注 / 控风险(6–8) → 协作 / 自动化放大(9–10) → 迭代与意义(11–12)**。
> 机制只当引擎,藏在底下。

---

## 第一幕 · 钱到底怎么回事

### 01 · 为什么你很努力,却不富有
- 努力是出售时间→线性、封顶;财富是复利→连乘、非线性。
- 富有的关键是资本积累:把流量(工资)变存量(会增值 / 产生现金流的资产)。
- 复利前期慢、后期陡;它怕两件事——本金太小、中途归零。
- 长期主义 = 给复利足够时间且不打断;穷忙 = 战术勤奋、战略懒惰。
- **引擎**:复利 · r>g(资本回报>劳动增长) · 存量vs流量。

### 02 · 你的劳动到底值多少钱
- 价格由供需定,不由辛苦定;你值多少 = 你做的事多稀缺、多难被替代。
- 你处在赚钱链条的哪一环:执行端(薄、易替代)还是判断 / 分发 / 资本端(厚)。
- 给资产定价的直觉:它未来能产生多少现金流、多确定(贴现)。
- 往价值更高的环节挪:从"按小时卖时间"走到"拥有定价权和资产"。
- **引擎**:供需定价 · 现金流贴现 · 价值链(微笑曲线)。

### 03 · AI 时代,什么在增值、什么在贬值
- 能被大量供给的塌价(初稿 / 画图 / 写代码);不能被供给的升值(判断 / 品味 / 验收 / 责任 / 关系)。
- 盘点你的技能组合:哪些在贬值(赶紧外包给 AI)、哪些在增值(加注)。
- 执行越便宜,它的互补品——"会定义问题、会验收"——越贵。
- **引擎**:供需 · 机会成本 · 互补品涨价。

## 第二幕 · 先把方向定对

### 04 · 你到底在为哪个目标努力
- 目标定错→越努力越偏(99% 准却把正样本全漏的模型)。
- 指标一旦变成目标本身,就会被你刷坏(Goodhart)。
- 先定义"什么算赢"(验收标准),再谈执行。
- 财务自由是约束、不是目标:现金流托底,让你不被迫在差选项里选。(自由意志的伏笔,12 回收)
- **引擎**:目标函数 · Goodhart · 约束优化。

### 05 · 个人战略:风口、趋势与影响力
- 战略 = 少而正确的卡位,不是多而勤奋的执行。
- 分辨噪声(热点)、坡度(趋势)、震荡(周期);押趋势,不押热点。
- 风口是顺势的杠杆——但别在看不懂的风口里当群众演员。
- 影响力 / 声誉是会复利的资产:一次建立,长期被反复调用。
- **引擎**:趋势vs噪声 · 卡位 · 声誉复利。

## 第三幕 · 下注与控风险

### 06 · 杠杆与试错:写一次运行很多次,先试一小口
- 杠杆 = 让一次判断被反复运行:代码、内容、资本、关系、AI。
- 试错 = 小步快跑、用小实验买反馈,不是学会了才开始。
- 好的试错:成本可控、可复盘、输有底赢没顶。
- **引擎**:固定成本摊薄 · 凸性 · 小步快跑。

### 07 · 别让大亏发生:仓位、配置与纪律
- 复利是连乘,一次大亏≈乘以接近零,前面全废——头等大事是不让大亏发生。
- all in vs 买大盘:集中是高风险结构,分散 / 指数是承认自己测不准;仓位决定你能不能继续玩。
- 怎么分配钱和时间:核心仓(稳、不归零)+ 卫星仓(高凸性的小注);时间也按这逻辑配。
- 纪律 + 执行:把规则(止损、仓位上限)在情绪上头之前预先设好,然后机械落地,不临场改。
- **引擎**:Kelly · 几何均值 · 资产配置 · 止损 / 预先承诺。

### 08 · 万一发生也别清零:风险、不确定性与鲁棒性
- 风险不是波动,是"不可逆的清零";分清可逆 / 不可逆,只对不可逆的极度小心。
- 未来测不准,所以靠鲁棒不靠预测:冗余、退路、不单点依赖、留可选项。
- 反脆弱:把自己放到能从波动里受益的一侧(凸性 / optionality)。
- 活下来是一切的前提:遍历性——你是顺时间走的那一个,碰一次归零就出局。
- **引擎**:不可逆风险 · 鲁棒 / 反脆弱 · 遍历性 · optionality。

## 第四幕 · 协作与自动化放大

### 09 · 与人协作,也和 Agent 协作
- 个人有上限,系统才有非线性;招人 / 委派 = 转移判断,不是甩任务。
- 讲不清目标和标准,招越多越累;价值观 = 组织的默认行为(正则项)。
- 和 Agent 协作:把判断封装成无人值守也能跑的流程。
- **引擎**:分布式>单机 · 转移判断 · Agent 自动化。

### 10 · 构建自动化、持续赚钱:品牌、人脉、粉丝
- 自动化 = 做一次、持续结算;把可重复的判断打包成 Skill / 流程。
- 品牌、声誉、粉丝、人脉是会自己滚的资产:信任降低每一次交易成本。
- 持续赚钱靠系统和资产,不靠你时时在场。
- **引擎**:规模化 · 信任资产 · 复利。

## 第五幕 · 迭代与意义

### 11 · 判断力:复盘、反馈与迭代
- 复盘 = 倒着追责:把结果翻译回"哪一步该改",让经历变成可复用的更新量。
- 及时反馈让判断越来越准;主动找能给你干净反馈的人。
- 拿到的结果(钱 / 数据 / 反馈)喂回认知,形成**认知复利**——回扣第 01 期。
- **引擎**:复盘 = credit assignment · 反馈 · 认知复利回流。

### 12 · 为什么要追求收益:钱是为了自由,不是为了钱
- **自由意志**:真正的自由是"为了想做而做",不是"为了逃避恐惧 / 匮乏而做"。出于逃避的努力,赚到了也不自由。
- 钱是手段不是目的:追求收益,是为了让你不必为钱牺牲注意力、自由、尊严——不必做不想做的事、看不想看的脸色、把时间卖给不认同的东西。
- 愿景越大、责任越大,越需要钱来支撑:钱不是终点,是你兑现愿景、扛起责任的弹药。
- 收官回扣:整季的"认知→复利→回报",最终是为了买回你的自由意志——让你能为了想做而做。这也是做这档节目的初心。
- **引擎**:钱 = 约束托底(回扣 04) · optionality = 选择权 = 自由 · 自由意志 vs 逃避动机。
